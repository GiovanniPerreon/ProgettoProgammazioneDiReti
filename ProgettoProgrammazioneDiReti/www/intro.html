Introduzione al progetto

Questo progetto mostra il funzionamento di un semplice server HTTP. Il server è in grado di:

  • Accettare richieste HTTP GET
  • Inviare file come .html, .css, immagini e altro
  • Restituire errori come 404 Not Found se il file richiesto non esiste

L'interfaccia web è composta da quattro pagine statiche:

  1. index.html: Home principale
  2. intro.html: Introduzione generale
  3. server.html: Come funziona il server
  4. header.html: Simulazione delle intestazioni HTTP
