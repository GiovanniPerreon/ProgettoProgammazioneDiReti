Come funziona il server HTTP

Questo progetto implementa un semplice server HTTP scritto in Python, il quale:

  • Ascolta richieste HTTP sulla porta 8080 del computer locale.
  • Riceve richieste GET da un browser o client HTTP.
  • Analizza la richiesta per determinare quale file statico serve (HTML, CSS, immagini, ecc.).
  • Verifica se il file richiesto esiste nella cartella www/.
  • Se il file esiste, lo invia al client con l’intestazione HTTP corretta (Content-Type basato sull’estensione del file).
  • Se il file non esiste, risponde con un messaggio di errore 404.

Il codice del server è contenuto nel file server.py e usa il modulo socket per gestire la comunicazione di rete.

Questa struttura permette di servire pagine web statiche semplici, simulando il funzionamento base di un vero web server.
